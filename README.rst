==================
HeroTerm README
==================

HeroTerm is a python based dropdown terminal made for the GNOME desktop environment.
HeroTerm's style of window is based on an FPS game, and one of its goals is to be easy to reach.

HeroTerm is a fork of `Guake Terminal <https://github.com/Guake/guake>`_, maintained by
HeroHubs. All credit for the original project goes to the Guake authors (see ``AUTHORS`` and the
About dialog). HeroTerm is distributed under the same license as Guake, the GNU GPL v2 or later.

Quick Installation Guide
------------------------

Please refer to the original `Installation Guide <https://guake.readthedocs.io/en/latest/user/installing.html#system-wide-installation>`_
(the build instructions still apply to this fork).

What it looks like ?
--------------------

Drop down terminal on pressing <F12>

Note when compiling from source
-------------------------------

Do **NOT** use the Tarball packages automatically generated by GitHub on the Release pages. They
won't work because one of the main components of the build system, PBR, requires the full Git
history to be available when building from source. Note this does not impact source distribution
packages.

Build from sources instructions are described on
`this page of the original Online Documentation <http://guake.readthedocs.io/en/latest/user/installing.html#install-from-source>`_.
Please read this carefully before opening an issue!
